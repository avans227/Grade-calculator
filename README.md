# Advanced Grade Calculator

A Python application that calculates the total, average, percentage, and final grade for multiple subjects.

## Features

- Accepts marks for multiple subjects
- Calculates total marks
- Calculates average marks
- Calculates percentage
- Assigns grades based on average marks
- Validates input marks (0–100)

## Grading Scale

| Marks Range | Grade |
|------------|--------|
| 85–100 | A+ |
| 75–84 | A |
| 65–74 | B+ |
| 55–64 | B |
| 45–54 | C+ |
| 35–44 | C |
| 30–34 | D |
| Below 30 | F |

## Example

Input:

Math: 90

Science: 80

English: 70

Output:

Total: 240

Average: 80.0

Percentage: 80.0%

Grade: A
